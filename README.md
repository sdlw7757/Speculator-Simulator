## 快速上手（10分钟通关攻略）
### 基础操作（超简单）
1. **买入**：商店页输入数量 → 点击「买入」（资金不足会提示）
2. **卖出**：仓库页输入数量 → 点击「卖出」（即时显示盈亏，赚差价超爽）
3. **推进**：点击「进入下个月」，坐等价格波动/突发事件（核心解压点）
4. **通关**：总资产到500万即可结束，全程仅需反复低买高卖，轻松达成

### 解压小技巧
- 不用纠结精准操作，随机事件带来的价格暴涨能快速拉高收益
- 重仓押注某一类商品，触发利好事件时一次性卖出，体验翻倍赚钱的爽感
- 即使破产也能一键重开，无任何惩罚，主打一个轻松玩
